# 📝 Markdown Previewer

A live Markdown editor and previewer built with React. Users can type GitHub-flavored Markdown (GFM) on the left and see the real-time rendered output on the right. Built for the freeCodeCamp Front End Libraries Certification project.

---

## 🚀 Features

- ✍️ Live preview of Markdown as you type
- 🪄 GitHub-flavored Markdown support
- 💻 Responsive, split-screen layout
- 🔍 HTML preview with safe rendering
- 🧹 "Clear" or "Reset" option

---

## 🛠️ Technologies Used

- React.js
- marked.js (Markdown parser)
- DOMPurify (to sanitize HTML)
- HTML5 + CSS3
- JavaScript (ES6+)

---

## ✅ User Stories (freeCodeCamp Spec)

- ✅ I can see a `textarea` element with `id="editor"`
- ✅ I can see a `div` element with `id="preview"`
- ✅ When I enter Markdown into the `editor`, the `preview` updates in real-time
- ✅ When I enter Markdown for headers, bold, italics, links, etc., they appear correctly rendered
- ✅ Markdown is rendered as HTML using a library like `marked`

---

## 🖱️ How to Use

1. Type your Markdown in the left-side editor.
2. View the real-time preview on the right.
3. Try using bold (`**bold**`), headings (`# Heading`), lists, code blocks, etc.
